Feature 1: File Upload

Users can upload images of their phone to the website to display any certain software issues or physical damage done to the phone to those who may not know how to explain the problem but are able to see that there is a problem.

Feature 2: Drag and Drop API

Users can drag and drop the phone they would like to be lended while their phone is being repaired or drop it back if they decided they no longer want to borrow a courtesy phone.

Feature 3: Local Storage

User preferences can be stored on the clients browser for the website to retrieve to personalise and improve the users experience.

Feature 4: SVG Image

Users can interact with an interactive map to help them locate one of the many stores located in New Zealand to find a local store that is the closest to reach.

Feature 5: Modal Form

Receipts can be printed on the client side to reduce the amount of work load that is done on the server side. This also allows for receipts to be kept on the client side when the user needs to print the receipt.
